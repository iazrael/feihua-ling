
# 飞花令项目配置

这是一个使用 Vue3 + TS 开发的飞花令H5应用, 运行在手机和平板上.

## 编码规范
- 使用 TypeScript 严格模式
- 遵循 ESLint 配置
- 注释必须用中文

## 架构说明
- 前端：Vue3 + TypeScript + Tailwind CSS
- 后端：Node.js + Express + Prisma

## 项目约定
- 架构分层合理, 易扩展
- API 路由统一前缀 `/api/v1/`
- 组件放在 `src/components/`
- 工具函数放在 `src/utils/`
- 测试文件使用 `.test.ts` 后缀
